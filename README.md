# Sequoia Region PCA — Mockup

Concept site for the Sequoia Region of the Porsche Club of America. Multi-page static build.

## Stack

- HTML, CSS, vanilla JS — no build step
- Events render from `events.json` via fetch
- Deployed on Vercel

## Updating the calendar

See `how-to-add-events.md`. Edit `events.json`, push, the site updates.
